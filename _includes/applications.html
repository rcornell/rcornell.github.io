Applications

Applications that I have developed are listed below.

Awayy - An application designed to foster world travel and new experiences by guiding you to the perfect destination. Uses Google Places API and Dark Sky API to match potential travel destinations to your specified temperature, time of year, budget, and interests.
ShareIn - Legacy codebase project, a social network for renting useful items from others in your community. Add things you own and want to share to your profile, then others nearby can search for them. For a fee, people can borrow these items from you.
Asset Management Platform - A C#/WPF-based desktop application to practice buying and selling equities, ETFs, and mutual funds. Uses Yahoo Finance API to track real time market data and position value.
SaveMonkey - An archiving tool for backing up video game save data on your PC. Uses Giant Bomb API and GamesDB API to populate game date. Can backup manually on request or at specific intervals.
